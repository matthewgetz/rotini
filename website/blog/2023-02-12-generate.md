rotini can now generate a javascript or typescript starting point for your cli program:  

```mdx-code-block
<Tabs>
<TabItem value="JavaScript">
```

```bash
npx rotini generate js
npm i rotini
node index.js
```

```mdx-code-block
</TabItem>
<TabItem value="TypeScript">
```

```bash
npx rotini generate ts
npm i rotini
ts-node index.ts
```

```mdx-code-block
</TabItem>
</Tabs>
```
